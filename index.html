Projects

PlaytestUrGame

Sep 2025 – Present

Developed a backend-driven MVP for indie game playtesting, designed to capture actionable player feedback and generate concise analytics while minimizing performance and storage overhead. The platform supports multiple games via a unified frontend, with robust user management and full CRUD functionality. Players can provide real-time feedback through keyboard shortcuts, which is intelligently aggregated to highlight genuine gameplay issues across multiple users, while filtering out isolated or spurious reports using a dynamic, runtime-adaptive algorithm. To minimize performance impact and storage requirements, selective screen recording captures critical gameplay segments for one player at a time. Build artifacts are currently stored via MinIO, leveraging its S3-compatible API within a containerized environment to allow runtime playback directly from storage. This approach simulates cloud-based storage workflows locally, providing DevOps-aware experience with artifact management and scalable infrastructure planning. In future iterations, the system is designed to integrate Cloudflare R2 object storage for fully cloud-hosted, S3-compatible build storage. The backend is modular, powered by Supabase for user and data management, and all services are containerized for reproducible local testing. The architecture is designed with extensibility in mind, including potential integration with payment systems and user storage tiers, enabling seamless expansion as the platform grows.
